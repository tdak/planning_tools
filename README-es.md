# Lista de recursos de planificación, gestión de proyectos y productividad

La planificación de proyectos es uno de los campos de más rápido crecimiento en el mundo. La planificación adecuada proporciona una aplicación valiosa hacia el logro de objetivos estratégicos. Aquí hay una lista de artículos, recursos y herramientas que puede usar para mejorar su productividad y habilidades de planificación.
